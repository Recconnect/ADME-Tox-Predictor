## 1. Ключевые инсайты из интервью

### 1.1. Почему биотех?
- **Purpose**: "Я в бизнесе переработки времени в счастье" — после $10M+ деньги
  перестают быть мотиватором, здоровье и лечение людей — главная ценность
- **Timing**: AI-дисрапшен фармы только начинается — "большинство больших
  бизнесов провалит AI-трансформацию, у маленьких компаний есть шанс"
- **Data**: "В биотехе много открытых данных, много денег, мало AI-специалистов"

### 1.2. Бизнес-модель BIOPTIC (3 источника дохода)

| Модель | Описание | Timeline |
|--------|----------|----------|
| **AI Agent (SaaS)** | "Cursor для фармы" — платят за токены | Immediate |
| **IP/Royalty** | Патенты на молекулы, продажа роялти-агрегаторам | 5-10 years |
| **M&A Exit** | Поглощение Big Pharma ($400M+) при достижении этапа | При валидации |

### 1.3. Почему маленькие побеждают (Innovator's Dilemma)
- Big Pharma: "first, do no harm" — риск-аверсия, медленные процессы
- Стартапы: "move fast, break things" — скорость в DNA
- Пример: BIOPTIC работает с теми же гигантами, которые могли бы быть
  конкурентами (платят миллиарды стартапам)
- "Все три компании из названных (Isomorphic, Pfizer) пришли к нам и стали
  работать"

### 1.4. Пайплайн Drug Discovery
```
Hit ID → Lead Optimization → Preclinical → Phase I → Phase II → Phase III
   |            |                 |           |         |          |
  AI:        AI:               AI:        AI:       AI:        AI:
 часы       недели            экспер.    экспер.   экспер.    экспер.
 $100K      $100-500K         $1-5M      $10M+     $50M+      $100M+
```

- **Ключевой инсайт**: Самый простой этап для AI — самый ранний (Hit ID +
  Lead Optimization), где сложность ограничена, а данных много
- **Именно здесь позиционируется ADME/Tox Predictor**

## 2. Позиционирование ADME/Tox Predictor

### 2.1. Где мы находимся в пайплайне

```
Hit ID  →  Lead Optimization  →  Preclinical → Clinic
              ↑                    ↑
         ADME/Tox Predictor     BIOPTIC etc.
         (наша ниша)            (их ниша)
```

**ADME/Tox Predictor** закрывает Lead Optimization:
- Предсказание ADME-свойств **до синтеза**
- Отсев «плохих» молекул на раннем этапе
- Экономия 60% бюджета R&D (именно столько уходит на Lead Optimization)

**BIOPTIC** закрывает Hit ID:
- Генерация новых активных молекул под мишень
- Поиск иголки в стоге сена (миллиарды виртуальных молекул)

### 2.2. Модель — не конкуренция, а комплемент

| Этап | Инструмент | Стоимость без AI | С AI |
|------|-----------|-----------------|------|
| Hit ID | BIOPTIC / Isomorphic | $100M, годы | $100K, часы |
| **Lead Optimization** | **ADME/Tox Predictor** | **$500M, 2-3 года** | **$50K, дни** |
| Preclinical | Wet lab + in vivo | $10-50M | $5-10M |

### 2.3. Уроки от BIOPTIC для нас

1. **Сервисная модель (SaaS/API)** — первичный доход
   - BIOPTIC: "Cursor для фармы" → мы: "API для ADME/Tox скрининга"
   - $10K-50K/год за лицензию — консервативная оценка

2. **Exit через M&A** — реалистичная стратегия
   - Наша ценность: данные + trained models на proprietary данных CRO
   - Потенциальные покупатели: Schrodinger, Charles River, LabCorp

3. **Royalty как финанс. инструмент** — на будущее
   - Если мы будем не только скринить, но и генерировать молекулы
   - Продажа роялти-агрегаторам (Royalty Pharma) при достижении milestone

4. **Target-agnostic подход** — ключевой манёвр
   - Наши модели работают на любых молекулах (не привязаны к белку)

5. **Data moat** —竞争优势
   - BIOPTIC использует публичные данные → все могут
   - Мы должны **собирать proprietary данные через партнёрства с CRO**
   - Это создаст барьер входа

## 4. Product Vision (в стиле Андрея Дороничева)

**"Мы в бизнесе переработки времени исследователей в скорость открытия
лекарств."**

- **Метрика**: не деньги, а "дни сэкономленного времени на одну молекулу"
- **Mission**: сократить Lead Optimization с 2-3 лет до 2-3 недель
- **Design Principle**: "одна молекула — один клик", никакой сложной химии
  в UI

## 5. Risks

| Риск | Вероятность | Митигация |
|------|------------|-----------|
| Big Pharma построит in-house | Средняя | Специализация + скорость |
| Данных недостаточно | Низкая (TDC открыт) | Fine-tuning на данных CRO |
| Регуляторные барьеры | Низкая (pre-clinical) | Не подпадает под FDA |
| Конкуренция (Schrodinger) | Средняя | Ниша ADME/Tox, цена в 10x ниже |
