📋 Инструкция по запуску:

  1. Запустите файл start_simple.bat (двойной клик)
  2. Дождитесь появления сообщения "You can now view your Streamlit app"
  3. Откройте браузер и перейдите по ссылке ниже
  4. Введите SMILES молекулы (например: CCO для этанола)
  5. Нажмите "Predict" для получения предсказаний
🚀 Открыть ADMETox.AI 🔄 Альтернативная ссылка (127.0.0.1)

🧪 Тестовые молекулы:

  • Этанол: CCO
  • Аспирин: CC(=O)Oc1ccccc1C(=O)O
  • Кофеин: CN1C=NC2=C1C(=O)N(C(=O)N2C)C
  • Ибупрофен: CC(C)Cc1ccc(cc1)C(C)C(=O)O

❓ Если не работает:

  1. Убедитесь, что start_simple.bat запущен
  2. Проверьте, что в окне cmd нет ошибок
  3. Попробуйте альтернативную ссылку (127.0.0.1)
  4. Проверьте, не заблокирован ли порт 8501 firewall
  5. Перезапустите start_simple.bat
