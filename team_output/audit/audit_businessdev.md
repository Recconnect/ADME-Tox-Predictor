## Ключевые находки

1. Рынок AI в drug discovery: global $3.5B (2025) → $7.6B (2028), CAGR 30%
2. Российский фармрынок: ~$30 млрд (2025), из них R&D — ~$1.5 млрд
3. Российские CRO тратят $50-200K/год на in silico инструменты (оценка)
4. TAM (РФ): $50-100 млн/год — AI ADME/Tox для CRO и фармы
5. Целевая доля: 2-5% рынка РФ = 5-15 клиентов при ARPU $10K/год = $50-150K ARR
6. Сегмент 1: CRO (ChemRar, Khimrar, PharmChem) — прямая потребность в скрининге молекул
7. Сегмент 2: Фармпроизводители (BIOCAD, R-Pharm, Geropharm) — R&D отделы
8. Сегмент 3: Биотех-стартапы РФ — мало, но быстрорастущий сегмент
9. Сегмент 4: Академические лаборатории — для free-tier и building brand
10. Всего целевых компаний в РФ: ~20 CRO + ~30 pharma R&D = 50 клиентов
11. Прямые конкуренты в РФ: отсутствуют — AI ADME/Tox ниша не занята
12. Непрямые: Schrodinger ($100K/год), ChemAxon ($5K/год) — не AI-native, дороги для РФ
13. Потенциальные: Insilico (глобально, нет dedicated ADME-продукта)
14. Академические: Way2Drug (PASS) — веб-инструмент, не SaaS, некоммерческий
15. Конкурентное преимущество: AI-native, российское происхождение, цена $10K/год
16. Риск: снижение цены на Schrödinger для РФ — но они фокусируются на крупных клиентах
17. Базовый тариф: $10K/год (₽800K/год) — 10,000 предсказаний, email support
18. Enterprise: $50K/год (₽4M/год) — dedicated instance, fine-tuning, API SLA
19. Free tier: 100 предсказаний/месяц — для академии и первичного знакомства
20. Credit-based: $50/1000 предсказаний — для разовых проектов
21. Цена в ₽ фиксированная: ₽60K/мес за enterprise — учтён курс и платежи РФ
22. Канал 1: Direct sales (LinkedIn, email, конференции) — основной для РФ
23. Канал 2: Партнёрства с вендорами (например, Cloud4Y для хостинга в РФ)
24. Канал 3: Госзакупки (ФЗ-44, 223) через статус участника Сколково
25. Канал 4: Реферальная программа для CRO-партнёров
26. Канал 5: Pharmaprojects каталог — внесение в реестр CRO-услуг
